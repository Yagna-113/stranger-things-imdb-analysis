# Stranger Things IMDb Analysis

This project analyzes IMDb ratings of Stranger Things episodes using Python and data science techniques.

## Dataset
- Episode ratings
- Runtime
- Genres

## Analysis Performed
- Season-wise rating trends
- Episode-wise rating analysis
- Runtime vs Rating correlation
- Top & lowest rated episodes
- Rating distribution

## Tools Used
- Python
- Pandas
- Matplotlib
- Kaggle Notebook

## Insights
- Later seasons have higher ratings
- Major plot episodes have higher audience engagement
- Runtime does not strongly affect rating
